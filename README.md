# SPLITR

SIDE PANEL BROWSING.

## FUNCTIONALITY

OPENS LINKS IN A SPLIT VIEW.
RESIZABLE CONTAINER.
PERSISTENT STATE.
HEADER BYPASS FOR IFRAME COMPATIBILITY.

## INSTALLATION

1. DOWNLOAD SOURCE.
2. NAVIGATE TO CHROME://EXTENSIONS.
3. ENABLE DEVELOPER MODE.
4. LOAD UNPACKED.
5. SELECT EXTENSION FOLDER.

## CONTROLS

- CLICK ACTION ICON: TOGGLE STATE.
- CLICK LINK: OPEN IN PANEL.
- DRAG EDGE: RESIZE PANEL.
- ESCAPE KEY: CLOSE PANEL.

## SYSTEM

REQUIRES PERMISSIONS TO MODIFY HEADERS AND INJECT SCRIPTS.
USES DECLARATIVE NET REQUEST API.
NO TRACKING. NO ANALYTICS.
